Installer ACRE2

1. Télécharger et installer TeamSpeak 3 si vous ne l'avez pas déjà : https://teamspeak.com/fr/downloads/

2. S'abonner au mod ACRE2 depuis le Workshop STEAM : https://steamcommunity.com/workshop/filedetails/?id=751965892

3. Activer le mod depuis le launcher, et lancer Arma 3 une première fois. Une fenêtre va apparaitre pour indiquer que l'extension a été installée dans TeamSpeak.

4. Lancer TeamSpeak ou Fermer et relancer TeamSpeak s'il était déjà lancé.

ACRE2 est configuré et prêt à l'utilisation.

Voir également: Guide d'installation (Anglais)

Utilisation de ACRE2

Le volume de la voix peut être modulé en maintenant appuyé la touche Tab ↹ avec la molette de la souris.

Lorsque vous êtes équipé d'une radio, le boutton pour parler est la touche Verr Maj (Caps Lock sur certains claviers).

Les réglages d'une radio se font via le menu ACE, c'est à dire en maintenant appuyé les touches Ctrl + Windows.

Les raccourcis et les paramètres de ACRE2 sont disponibles dans les menus standards des addons.

Résolution des problèmes fréquents

Message "ACRE2 Installation Error"

Si le message précédent apparait, fermer TeamSpeak, puis cliquer sur "Recommencer". Attendre que Arma 3 finisse de démarrer et relancer TeamSpeak.

Mauvaise version du plugin TeamSpeak

L'installation du plugin TeamSpeak est automatique, mais elle peut parfois échouer. Dans ce cas il faut procéder à une installation manuelle.

1. Arrêter TeamSpeak

2. Ouvrir le répertoire C:\Program Files (x86)\Steam\steamapps\common\Arma 3\!Workshop\@ACRE2\plugin avec l'explorateur de fichier (le répertoire peut être différent si Arma 3 a été installé sur un autre disque dur).

3. Copier les fichiers acre2_win32.dll et acre2_win64.dll (Ctrl + C).

4. Aller dans le répertoire %USERPROFILE%\AppData\Roaming\TS3Client\plugins avec l'explorateur de fichier.

5. Coller les fichiers (Ctrl + V), et choisir l'option "Remplacer les fichiers".

6. Relancer TeamSpeak

Remarque: Pour ouvrir l'explorateur sur les emplacements sur les répertoires précisés, vous pouvez utiliser le raccourcis Windows. + R, coller la chemin et valider.
